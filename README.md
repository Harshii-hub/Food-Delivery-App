🍔 Food Delivery App — README
📌 Overview
This Food Delivery App is a modern platform that allows users to browse restaurants, explore menus, place orders, and get food delivered to their doorstep. It is designed to provide a smooth, fast, and user-friendly experience for both customers and restaurant partners.

🚀 Features
👤 User Features
User registration & login (authentication)

Browse restaurants and menus

Search and filter food items

Add items to cart

Secure checkout process

Real-time order tracking

Order history & reordering
